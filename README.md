# startcss

Startcss is a simple css framework built with sass,postcss,lost.

## Installation

**NPM**: install with npm to package.json

`npm install stratcss -S`

**Download Manually**

Simply download or clone this repository

## Structure

Folder structure of `Build`:

```
├── css
│   │
│   └── startcss.css
├── index.html
└── js
    ├── main.js
    └── maps
        └── main.js.map
```
